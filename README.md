folder based static website and image gallery viewer
 
# Usage
 
 * `git clone https://github.com/hcnode/SamGallery`
 * `npm install` 
 * `vi ./config/default.json` config the "homePath"
 * `npm start`
 * Visit http://localhost:8000/
